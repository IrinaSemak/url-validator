## Как запустить лабораторную за 2 минуты  
(даже если у тебя ничего не установлено)

1. Перейди по ссылке:  
   https://github.com/IrinaSemak/url-validator

2. Нажми зелёную кнопку **Code** → **Download ZIP**

3. Распакуй архив в любую папку (например, на Рабочий стол)

4. Открой распакованную папку

5. Найди файл **Запуск.bat**  
   (у него значок шестерёнки или чёрного окошка)

6. **Дважды щёлкни по нему левой кнопкой мыши**

7. Подожди 10–20 секунд — всё скачается и запустится автоматически

8. При первом запуске появится сообщение, что не хватает файла `app.properties`  
   → Это нормально! Сделай вот что (только один раз):

   **Создай конфигурационные файлы за 10 секунд:**

   - В корневой папке проекта создай файл **`app.properties`**  
     (правой кнопкой → Создать → Текстовый документ → переименуй в `app.properties`)
   - Открой его и вставь точно такое содержимое:

     ```properties
     urls.file.path=config/urls.txt
     http.method=HEAD
     http.timeout.ms=5000

   **Создай папку config (правой кнопкой → Создать → Папку)

   - Внутри папки config создай файл urls.txt и вставь любые ссылки, например:

https://github.com
https://google.com/no-such-page-404
https://httpstat.us/503
https://httpstat.us/200


9. Снова дважды щёлкни по файлу Запуск.bat
